# Support

## Getting help

If you run into a problem with Doc Gen Studio, please start here before opening an issue.

1. **Check the existing issues** — search [open and closed issues](https://github.com/presdec/docgen-studio-desktop/issues?q=is%3Aissue) to see if your problem has already been reported or resolved.
2. **Open a bug report or feature request** — use the [issue templates](https://github.com/presdec/docgen-studio-desktop/issues/new/choose) to give enough context for a quick response.
3. **Email** — for questions that don't fit a GitHub issue, reach out directly at [presdec+docgen@gmail.com](mailto:presdec+docgen@gmail.com).

## Response times

This is an open-source project maintained in spare time. Responses are best-effort, typically within a few days.

## Security issues

Please **do not** open a public issue for security vulnerabilities. Email [presdec+docgen@gmail.com](mailto:presdec+docgen@gmail.com) directly instead.
